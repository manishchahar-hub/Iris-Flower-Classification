# Iris Flower Classification

This project is a Machine Learning based Iris Flower Classification system that predicts the species of an iris flower based on different flower measurements.

The model is built using Logistic Regression and trained on the Iris dataset.

---

## Features

- Iris flower species classification
- Data preprocessing and feature engineering
- Categorical data encoding using LabelEncoder
- Feature scaling using StandardScaler
- Logistic Regression classification model
- Data visualization using different plots
- Outlier detection using Boxplot
